# Learsim Control Panel
</div>

## Purpose
Learsim Control Panel for managing mostly Arduino clients for FS2020. 
Using [Learsim Server](https://github.com/Learsim/LearsimServer).

## Usage
W.I.P

## Development 

#### Clone repo
```git clone https://github.com/Learsim/ControlPanel.git```
#### Go to directory
```cd ControlPanel```
#### Run yarn
```yarn```
#### Run yarn start
```yarn start```
